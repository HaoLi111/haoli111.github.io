## Machine Learning & AI

### sk_in_torch
A PyTorch implementation for fitting scikit-learn models with GPU/CPU support.

**Repository**: [github.com/HaoLi111/sk_in_torch](https://github.com/HaoLi111/sk_in_torch)

**Technologies**: Python, PyTorch, scikit-learn

**Description**: This project bridges the gap between PyTorch and scikit-learn, allowing you to leverage GPU acceleration for traditional ML algorithms.
